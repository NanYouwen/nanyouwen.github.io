Less(css预处理器)详解

Less介绍

​ Less 是一种动态样式语言，扩展了 CSS 语法，支持变量、嵌套、混合、函数等特性。利用 Less，可创建更具可维护性、可读性的样式代码。

Less 的安装

​ 项目中安装 Less 可通过 npm 或直接使用 CDN 引入：

变量:利用变量定义颜色、字体等样式属性，便于全局修改。

嵌套:支持层级嵌套，使代码结构更清晰。

混合（Mixins）:使用混合创建可复用的样式块。

运算:支持算术运算，便于动态调整样式。

媒体查询:支持嵌套的媒体查询写法。

颜色函数:内置多种颜色处理函数。

链接伪类:简洁的伪类写法。

嵌套列表:嵌套写法使得结构更清晰。
